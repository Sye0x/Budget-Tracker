# Budget Tracker

A simple and clean web app to track your monthly expenses. You can add expenses, edit them, delete them, and view your spending through a category-wise pie chart.

---

## Features
- Add new expenses  
- Edit and delete existing expenses  
- Set your monthly budget  
- Shows total spent and remaining amount  
- Visual spending breakdown using Chart.js  
- Saves all data using localStorage  

---

## Built With
- **HTML** – structure  
- **CSS** – styling  
- **JavaScript** – app logic  
- **Chart.js** – pie chart visualization  
- **localStorage** – save data in browser  

---

## What I Learned
- How to store and manage data using localStorage  
- Creating dynamic UI updates with JavaScript  
- Building a CRUD system (add, edit, delete items)  
- Working with Chart.js to display data visually  
- Designing a modern and responsive layout with CSS
